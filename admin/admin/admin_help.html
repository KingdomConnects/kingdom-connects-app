Admin – Help & Support

Common Admin Tasks:

  • ✔️ Approve or reject pending business listings
  • ✔️ Approve or reject church submissions
  • ✔️ Moderate user reviews
  • ✔️ Check site activity logs
  • ✔️ Review system error logs if needed

Pro Tips:

  • ✅ Always double-check before approving listings
  • ✅ Keep an eye on the activity log for suspicious patterns
  • ✅ When in doubt, leave a note in the system logs for future admins
